Flexbox Teaching Example

This example demonstrates key Flexbox properties. Hover over each item to learn about the properties applied.

Item 1
Item 2
Item 3
Flex Container Properties:
display: flex;
flex-direction: row;
justify-content: space-between;
align-items: center;
Flex Item Property:
flex: 1; /* Items grow equally */

Explanation:

  • display:flex: Enables flex layout for container.
  • flex-direction: Sets main axis (row or column).
  • justify-content: Aligns items along main axis (horizontal here).
  • align-items: Aligns items along cross axis (vertical here).
  • flex:1: Makes each item grow equally to fill container space.
